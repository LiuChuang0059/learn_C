# 用于迭代器

* 因为很多迭代器不支持算术运算，递增递减必须。

# 前置版本------建议使用
运算符将对象加一，然后将改变后的对象作为求值结果。
```cpp
int i =0 ,j;
j= ++i;   // i=1 j=1.
j=i++; // j=1 ,i=2,,,求值结果是改变对象前的一个副本
```
